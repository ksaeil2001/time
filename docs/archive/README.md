# Archive Policy

## Purpose
- 이 디렉터리는 정본 문서의 이전 버전을 보존하기 위한 공간이다.
- 목적은 회귀 분석, 감사 추적, 버전 간 비교를 가능하게 하는 것이다.

## Naming Rules
- 폴더명은 다음 형식을 따른다.
  - `YYYY-MM-DD_pre-refactor`
  - `v0.x` (릴리즈 기준 보관 시)
- 한 폴더에는 같은 시점의 관련 문서를 함께 보관한다.
